About
========================
This module allows customers send order request for out of stock products.
Instead of "Add to cart" customer will see "Order product" button.
On submit record will be added to database and on cron execution
summary to all orders sends to store email.
Suppors product attributes and anonymous orders.


Installation
========================
After module activation set permissions at admin/user/permissions.


Описание
========================
Модуль позволяет покупателям отправлять запросы на заказ товаров, отсутствующих
на складе. Вместо кнопки "Добавить в корзину" покупатель увидит "Заказать товар".
При нажатии на нее будет добавлена запись в базу данных, и при выполнении cron
письмо с суммарными данными будет отправлено на почтовый ящик магазина.
Поддерживает атрибуты товаров и анонимные заказы.

Установка
========================
После активации модуля настройте права на admin/user/permissions.
